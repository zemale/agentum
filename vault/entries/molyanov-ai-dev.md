## Описание

Публичный репозиторий с методологией агентной разработки, выстраданной за 5 месяцев работы с Claude Code. Скиллы и агенты для ежедневной работы с проектами.

## Суть методологии

1. **project-knowledge** — в каждом проекте собирается вся документация, обновляется после каждого цикла разработки
2. **shared/** — шаблоны папок проектов, артефактов
3. **Spec Driven Development:**
   - `user-spec-planning` — агент проводит интервью, пишет user-spec (план простыми словами)
   - `tech-spec-planning` — глубокое изучение кода, подробный план для агентов
   - `task-decomposition` — разбивка tech-spec на атомарные задачи (каждая в отдельной сессии с чистым контекстом)
   - `do-task` — ручной режим: одна задача за один чат
   - `do-feature` — ваншот-режим: agent team делает все задачи сразу
4. **skill-master** — создание своих скиллов и агентов-проверяльщиков
5. **skill-tester** — экспериментальный тестировщик скиллов

## Особенности

- На каждом этапе — проверки субагентами для минимизации ошибок
- Жрёт много токенов, без подписки max200 лимиты выгорают быстро
